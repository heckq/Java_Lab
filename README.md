# Minesweeper Game

## Опис
Це проект гри "Сапер", написаної на Java. Гра "Сапер" - це класична комп'ютерна гра, де гравці повинні відкривати клітинки, уникаючи мін.

## Встановлення
1. Клонуйте репозиторій за допомогою `git clone`.
2. Відкрийте проект у вашому улюбленому редакторі коду (наприклад, IntelliJ IDEA).
3. Запустіть клас Minesweeper, який знаходиться в пекеджі Main.

## Використання
Гра починається з того що гравця просять назвати розмір поля, і кількість мін на ньому.
Гравець відкриває клітинки, пишучи її координати на полі, і команду free.
Якщо під клітинкою міна - гра закінчується. 
Якщо під клітинкою міни немає, то відображається число, 
що показує, скільки мін розташовано навколо цієї клітинки, або якщо мін поблизості немає, відкривається певна частина поля. 
Мета гри - позначити всі клітинки під якими знаходяться міни за допомогою команди mark.
