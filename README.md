# FileLister

FileLister - це програма на Java, яка дозволяє користувачеві знайти файли з певним розширенням в заданій директорії та зберегти список назв файлів в інший файл.

## Встановлення

Для запуску програми вам потрібно мати встановлену Java на вашому комп'ютері. Ви можете завантажити Java [тут](https://markdown.land/readme-md).

## Використання

Для запуску програми виконайте наступні кроки:

1. Склонуйте або завантажте цей репозиторій на ваш комп'ютер.
2. Відкрийте термінал або командний рядок і перейдіть до директорії, де знаходиться файл FileLister.java.
3. Скомпілюйте файл за допомогою команди `javac FileLister.java`.
4. Запустіть програму за допомогою команди `java FileLister`.
5. Введіть шлях до директорії, в якій ви хочете знайти файли з певним розширенням.
6. Введіть розширення файлів, які ви хочете знайти (без крапки).
7. Введіть, чи ви хочете включити розширення файлу в список (yes/no).
8. Введіть шлях до файлу, в який ви хочете зберегти список.
9. Перевірте файл, в який ви зберегли список, і переконайтеся, що він містить правильні дані.

## Приклад

Ось приклад виводу програми, якщо ви введете наступні дані:

- Шлях до директорії: C:\Users\user\Documents
- Розширення файлів: txt
- Включити розширення файлу в список: yes
- Шлях до файлу, в який зберегти список: C:\Users\user\Desktop\list.txt
